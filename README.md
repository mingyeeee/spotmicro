# Spotmicro
A 3D printed quadruped for the purpose of experimenting with legged locomotion. 

Spotmicro build from https://www.thingiverse.com/thing:4559827.

## Hardware 
- Teensy 4.0 as main processor
- MPU 6050 IMU
- JX PDI HV5523 high voltage servos
